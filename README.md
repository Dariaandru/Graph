# Графы

## Структура данных - иерархический список.

### Граф - фигура которая состоит из вершин (точек) и ребер (отрезков) соединяющих их.

### Схема графа

Вершины - это города, ребра - дороги между ними. Числа - протяженность между городами.

![](./images/схема.png)
